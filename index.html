Terminology and Definition

  1. What do HTML and CSS stand for?

    2. HTML (HyperText Markup Language) is the code that is used to structure a web page and its content

  2. Would you rather use HTML or CSS, for putting paragraphs of text on a webpage?

    3. We use HTML for putting paragraphs of text on a webpage

  3. Would you rather use HTML or CSS, to change the font and background color of a button?

    4. We use CSS to change the font and background color of a button

  4. What is an HTML tag?

    5. An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document

  5. What are the three parts of an HTML element? (image from paragraph)

    6. An HTML element has opening tag, content and closing tag.

  6. What is the purpose of the doctype declaration?

    7. The doctype purpose is to tell the browser what version of HTML it should use to render the document.

  7. What is the HTML element?

    8. An HTML element is an individual component of an HTML document.

  8. What is the purpose of the head element?

    9. The element is a container for metadata (data about data) and is placed between the tag and the tag. HTML metadata is data about the HTML document. Metadata is not displayed. Metadata typically define the document title, character set, styles, scripts, and other meta information.

  9. What is the purpose of the body element?

    10. The tag defines the document's body. The element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. Note: There can only be one element in an HTML document.

  10. How do you create a paragraph in HTML?

    11. We type the paragraph start tag. Then type the paragraph's text. At the end of the new paragraph's text, we type the end tag.

  11. How do you create a heading in HTML?

    12. We use any of the heading tags and every tag will give us the different size of the heading.

  12. How many different levels of headings are there and what is the difference between them?

    13. HTML defines six levels of headings. A heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading. The heading elements are H1, H2, H3, H4, H5, and H6 with H1 being the highest (or most important) level and H6 the least.

  13. What element should you use to make text bold and important?

    14. The HTML b element defines bold text

  14. What element should you use to make text italicized to add emphasis to it?

    15. To italicize the text in HTML, use either the em tag or the i (italics) tag. Both of these tags will italicize the text, but the em tag additionally indicates that the text has stress emphasis when read.

  15. What relationship does an element have with any nested elements within it?

    16. When we nest elements within other elements, we create a parent and child relationship between them. The nested elements are the children and the element they are nested within is the parent.

  16. What relationship do two elements have if they are at the same level of nesting?

    17. Elements at the same level of nesting are considered to be siblings

  17. How do you create HTML comments?

    18. To comment out in HTML, we insert information between < ! -- and -- > tags

  18. What HTML tag is used to create an unordered list?

    19. The < ul > HTML element represents an unordered list of items

  19. What HTML tag is used to create an ordered list?

    20. The < ol > HTML element represents an ordered list of items

  20. What HTML tag is used to create list items within both unordered and ordered lists?

    21. The < li > HTML element create list items within both unordered and ordered lists

  21. What element is used to create a link?

    22. The < a > HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

  22. What is an attribute?

  23. What attribute tells links where to go?

    24. The hypertext reference, or href , attribute is used to specify a target or destination for the anchor element.

  24. What is the difference between an absolute and relative link?

    25. The main difference between absolute and relative paths is that absolute URLs always include the domain name of the site with http://www. Relative links show the path to the file or refer to the file itself. A relative URL is useful within a site to transfer a user from point to point within the same domain.

  25. Which element is used to display an image?

    26. The < img > element is used to display an image.

  26. What two attributes do images always need to have?

    27. The img element has two required attributes: src : The source location (URL) of the image file. alt : The alternate text. This is used to describe the image for someone who cannot see it because they are either using a screen reader or the image src is missing.
